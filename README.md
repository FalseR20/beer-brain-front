# BeerBrain - Front-end

BeerBrain front-end server uses React with React-Bootstrap css framework.

## Quickstart

Install dependencies:

```shell
yarn install
```

Run frontend server:

```shell
yarn run dev
```

For production:

```shell
yarn run build
yarn run preview
```

Docker:

```shell
docker build . -t "beer-brain-front"
docker run -p 4173:80 beer-brain-front
```
